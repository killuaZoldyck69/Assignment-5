Important FAQ

What is a Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of nodes. It allows scripts to dynamically access, modify, and update the document's content, structure, and style in real-time.

How do you select an element from the DOM?

document.getElementById('id') - Selects an element by its ID.
document.querySelector('selector') - Selects the first element that matches a CSS selector.
document.getElementsByClassName('class') - Selects all elements with a specific class.

What is event delegation in the context of the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events from multiple child elements. Instead of adding separate event listeners to each child, you leverage the event bubbling mechanism to capture events at the parent level and then determine which child element triggered the event. This approach improves performance and simplifies code management.

How do you manipulate an element's attributes and style using the DOM?

Attributes: Use setAttribute('attribute', 'value') or direct property assignment like element.id = 'newId'.
Style: Modify the style property, e.g., element.style.color = 'blue' or element.style.display = 'none'.
